# API Documentation

The Evolve node exposes a gRPC-Web compatible API over HTTP/1.1 with JSON encoding at `http://localhost:7331` by default.

## Available Services

- **Signer Service** — Transaction signing operations
- **Store Service** — Block and state store access
- **P2P Service** — Peer-to-peer networking
- **Config Service** — Node configuration
- **Health Service** — Health checks and status

Browse the API operations in the sidebar to see detailed request/response schemas for each endpoint.
